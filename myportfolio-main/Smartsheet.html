Exemple :
Les données des inspections navales sont collectées depuis des fichiers Excel et synchronisées automatiquement dans une feuille SmartSheet pour un suivi centralisé.

2. Automatisation des processus C5

Fonctionnalités : Automatisation des workflows.
  • Synchronisation automatique des données entre les fichiers externes et SmartSheet.
  • Réduction des erreurs humaines grâce à des processus standardisés.
  • Notifications automatiques pour informer les équipes des mises à jour.
Détail technique :
Les workflows Data Shuttle permettent de charger automatiquement les données des fichiers CSV dans SmartSheet, tout en appliquant des règles de validation pour garantir leur fiabilité.

Fonctionnalités : Dashboards dynamiques pour la prise de décision.
  • Suivi des indicateurs clés (KPI).
  • Rapports personnalisés pour chaque équipe.

Exemple :
Un dashboard interactif affiche les délais de maintenance, les coûts associés, et les performances des équipes, permettant aux managers de prendre des décisions éclairées.

Fonctionnalités : Suivi des demandes et collaboration via SmartSheet.
  • Création d'une feuille dédiée pour le suivi des bugs, mises à jour et ajouts.

🗂️ Script d'archivage et Docker

📜 Script d'archivage

Le script Python développé pour SmartSheet automatise l'archivage des projets clos. Voici les étapes principales :

  • Identification des lignes marquées comme "Projet Clos" dans la feuille source.
  • Transfert des lignes identifiées vers une feuille d'archive dédiée.
  • Nettoyage des données dans la feuille source pour maintenir une base propre.
  • Exécution automatique à intervalles réguliers grâce à un déploiement sur Docker.

Ce processus garantit une gestion efficace des données et réduit les erreurs humaines. 

# Exemple de code Python pour l'archivage
import smartsheet
from datetime import datetime

# Initialisation du client SmartSheet
smartsheet_client = smartsheet.Smartsheet('API_KEY')
smartsheet_client.errors_as_exceptions(True)

# Fonction principale d'archivage
def archive_lines():
    source_sheet_id = 123456789  # ID de la feuille source
    archive_sheet_id = 987654321  # ID de la feuille d'archive

    # Récupération des lignes marquées comme "Projet Clos"
    source_sheet = smartsheet_client.Sheets.get_sheet(source_sheet_id)
    rows_to_archive = [
        row for row in source_sheet.rows if row.cells[0].value == "Projet Clos"
    ]

    # Préparation des lignes pour l'archivage
    rows_to_add = []
    for row in rows_to_archive:
        new_row = smartsheet.models.Row()
        new_row.to_bottom = True
        new_row.cells = row.cells
        rows_to_add.append(new_row)

    # Ajout des lignes à la feuille d'archive
    smartsheet_client.Sheets.add_rows(archive_sheet_id, rows_to_add)

    # Nettoyage des lignes dans la feuille source
    for row in rows_to_archive:
        for cell in row.cells:
            cell.value = None
    smartsheet_client.Sheets.update_rows(source_sheet_id, rows_to_archive)

# Exécution du script
archive_lines()

🐳 Docker

Docker a été utilisé pour déployer le script d'archivage de manière automatisée et fiable. Voici les avantages de son utilisation :

  • Isolation de l'environnement d'exécution pour éviter les conflits de dépendances.
  • Déploiement facile sur différents serveurs ou machines.
  • Planification des tâches grâce à des outils comme cron ou des orchestrateurs comme Kubernetes.

Le script est encapsulé dans un conteneur Docker, ce qui permet une exécution régulière et fiable.
